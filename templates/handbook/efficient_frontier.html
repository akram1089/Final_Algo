Efficient Frontier

The Efficient Portfolio Frontier, introduced by Harry Markowitz in 1952, is a portfolio theory that rates investments in terms of return relative to risk.
View risk disclosures

The efficient portfolio frontier is a concept used in modern portfolio theory to identify the most profitable and least risky investment portfolios. It is based on the principle that an investor should seek to maximize returns while minimizing risk and consists of a set of optimal portfolios that offer the greatest expected return for any given level of risk.
At its core, the efficient portfolio frontier is founded on the notion that portfolio diversification can help to reduce portfolio risk without sacrificing return. This is achieved by combining assets with different levels of correlation, which results in more stable and less volatile returns over time.
By identifying and investing in assets along this frontier, investors can maximize the expected return of their portfolio while keeping risk at a minimum.

How to read the efficient frontier graph

The efficient frontier graph is a visual representation of the efficient portfolio frontier. It plots the expected return on the x-axis and risk (measured as standard deviation) on the y-axis. The resulting curve shows all possible portfolios that can be constructed with a given set of assets, with each point on the curve representing a different portfolio.

The x-axis represents return, with portfolios to the right offering higher expected returns. The y-axis represents risk, with portfolios to the top offering higher levels of risk. The efficient frontier is the curve that plots all possible portfolios that offer the highest expected return for any given level of risk.

Points that lie above the curve represent portfolios that offer a higher expected return than the efficient frontier, while those below the curve represent portfolios with lower expected returns. In general, the further away from the curve a portfolio lies, the greater its risk relative to its expected return.

Using the efficient portfolio frontier

There are several ways investors can use the efficient portfolio frontier to improve their portfolio management strategies.
For example, portfolio managers may seek to identify and capture the highest expected risk-adjusted return for each level of portfolio risk, or they may opt to construct portfolios with similar levels of portfolio risk but with lower minimum variance. FAQ's

How do you calculate the efficient frontier?

The Efficient Portfolio Frontier takes every possible combination of portfolio asset configuration and plots their risk versus return. Combinations of the risk-free rate, domestic equities, international equities, and other asset classes are considered to determine the optimal portfolio composition for a given level of risk.
